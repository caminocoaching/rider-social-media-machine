# FACEBOOK ALGORITHM 2026: COMPLIANCE GUIDE
## For Driver Coaching Content

---

## Overview

This guide ensures all Facebook content maximizes organic reach while avoiding algorithmic penalties. The 2026 algorithm prioritizes meaningful interactions over vanity metrics.

---

## 1. RANKING SIGNALS (What Facebook Rewards)

### Primary Signals

| Signal | Weight | How to Trigger |
|--------|--------|----------------|
| **Comments** | Very High | Ask specific questions; use comment CTAs |
| **Comment Replies** | Very High | Reply to every comment within 1 hour |
| **Saves** | High | Provide actionable, reference-worthy content |
| **Shares** | High | Create emotionally resonant content |
| **30+ Second Video Watch** | High | Hook in first 3 seconds; deliver value throughout |
| **Relationship Signal** | High | Engage with followers' content; build connection |

### Secondary Signals

| Signal | Weight | How to Trigger |
|--------|--------|----------------|
| **Reactions** | Medium | Emotional content (not engagement bait) |
| **Profile Clicks** | Medium | Establish authority and curiosity |
| **Time on Post** | Medium | Longer, valuable content |

---

## 2. CONTENT PENALTIES (What to Avoid)

### Immediate Demotion Triggers

| Violation | Example | Why It's Penalized |
|-----------|---------|-------------------|
| **Engagement Bait** | "Like if you agree!" | Manipulative; inflates metrics artificially |
| **Tag Bait** | "Tag a driver who needs this" | Spam-like behavior |
| **Share Bait** | "Share with your paddock friends" | Manipulation of share metric |
| **External Links** | Link in main caption | Takes users off-platform |
| **Clickbait** | "You won't believe what happened next" | Misleading; poor user experience |

### Gradual Demotion Triggers

| Behavior | Impact | Alternative |
|----------|--------|-------------|
| Inconsistent posting | Reduced reach over time | Post daily at consistent times |
| No engagement with comments | Weakens relationship signal | Reply within 1 hour |
| Identical content across posts | Flagged as spam | Vary format and approach |
| Overly promotional | Lower distribution | 80% value, 20% offer |

---

## 3. SAFE PHRASES VS. PENALIZED PHRASES

### ✅ Safe CTAs

| Phrase | Why It Works |
|--------|--------------|
| "Drop a [WORD] below" | Specific action, not generic engagement |
| "Comment [WORD] if you want this" | Self-selection, not manipulation |
| "If this sounds familiar, comment [WORD]" | Invites genuine response |
| "Type [WORD] and I'll send it over" | Clear value exchange |
| "Does this resonate? Let me know below" | Open conversation |
| "What's your experience with this?" | Genuine question |

### ❌ Penalized CTAs

| Phrase | Penalty Type |
|--------|--------------|
| "Like this post if you agree" | Engagement bait |
| "Share with someone who needs to hear this" | Share bait |
| "Tag a friend" | Tag bait |
| "Link in bio" | Off-platform redirect |
| "Click the link below" | External link penalty |
| "Double tap if you relate" | Engagement bait |
| "Comment YES if you want more" | Generic engagement bait |

## 4. POST FORMATTING FOR MOBILE

### The 81% Rule
81% of Facebook users access via mobile. All content must be mobile-optimized.

### Formatting Guidelines

| Element | Guideline |
|---------|-----------|
| **Paragraphs** | 1-3 sentences maximum |
| **Line Breaks** | Use liberally between ideas |
| **Emojis** | 1-2 per post maximum; never in opening line |
| **ALL CAPS** | Avoid entirely (except trigger words) |
| **Bold/Italic** | Not available in organic posts |
| **Character Count** | Aim for 150-300 words |
| **First Line** | Must hook immediately (truncated in feed) |

### Visual Structure Example

```
Short punchy opening line.

Second idea here.
Expand on it briefly.

Third point.

Maybe a question here?

Your CTA.
```

### What NOT to Do

```
This is a long paragraph with multiple ideas crammed together that forces the reader to work hard to extract meaning and will cause them to scroll past because walls of text are exhausting on mobile devices and nobody has time for this especially when they're scrolling in the paddock between sessions.
```

## 5. POSTING SCHEDULE

### Optimal Times (UK)

| Day | Time | Content Type |
|-----|------|--------------|
| Monday | 7:00 AM | Problem/Hook post |
| Tuesday | 12:00 PM | Pain agitation |
| Wednesday | 7:00 AM | Neuroscience/Education |
| Thursday | 6:00 PM | Future pacing |
| Friday | 7:00 AM, 12:00 PM, 6:00 PM | Handraiser variations |

### Why These Times Work

- **7:00 AM:** Morning scroll before the day starts
- **12:00 PM:** Lunch break engagement
- **6:00 PM:** Evening wind-down

### Consistency > Frequency

The algorithm rewards consistent posting more than frequent posting. Better to post daily at the same time than sporadically throughout the day.

## 6. ENGAGEMENT PROTOCOL

### Comment Response Rules

| Timeframe | Action | Why |
|-----------|--------|-----|
| 0-60 minutes | Reply to all comments | Signals active engagement to algorithm |
| 1-4 hours | Continue replying | Maintains post momentum |
| 24+ hours | Still reply | Shows community care |

### Response Types

| Comment Type | Response Strategy |
|--------------|-------------------|
| Trigger word only | DM immediately; acknowledge in comment |
| Personal story | Validate, ask follow-up question |
| Question | Answer thoroughly |
| Skepticism | Address honestly, don't get defensive |
| Praise | Thank them; ask what resonated |

### Example Responses

**Trigger Word Comment:**
> "MINDSET"

**Response:**
> "Sent! Let me know what clicks for you—I read every reply 🏁"

**Story Comment:**
> "This is exactly what happened to me at Brands last weekend"

**Response:**
> "Brands can be brutal for this. Which corner was hitting you hardest?"

## 7. LINK HANDLING

### The Rule
**Never put links in the main post caption.**

### Approved Methods

| Method | When to Use |
|--------|-------------|
| **Comment CTA → DM** | Free training offers (recommended) |
| **First Comment** | If link must be public (lower reach) |
| **Profile Bio** | Evergreen links only |

### The Two-Step Process

1. **Post:** "Comment [WORD] and I'll send it over"
2. **DM:** Send link privately with personalization

### Why This Works Better

- No algorithmic penalty for external links
- Creates personal connection
- Allows conversation before clicking
- Higher conversion rate than cold links

## 8. VIDEO CONTENT RULES

### Reels (Short-Form)

| Element | Guideline |
|---------|-----------|
| **Length** | 15-60 seconds optimal |
| **Hook** | First 3 seconds must grab attention |
| **Format** | Vertical (9:16) |
| **Captions** | Always include (sound-off viewing) |
| **Audio** | Original audio preferred over trending |

### Long-Form Video

| Element | Guideline |
|---------|-----------|
| **Length** | 3-15 minutes |
| **Watch Time** | Optimize for 30+ second retention |
| **Native Upload** | Never post YouTube links |
| **Thumbnails** | Clear, high contrast, readable text |

### Video Hooks That Work

✅ "Here's why your qualifying pace disappears..."
✅ "The mistake I see P4 drivers make..."
✅ "In 2006, Stanford discovered something about drivers..."

❌ "In this video I'm going to talk about..."
❌ "Hey guys, so today I wanted to share..."
❌ "Before I get started, make sure to like and subscribe..."

## 10. RECOVERY FROM LOW REACH

If reach has collapsed:

### Immediate Actions (Days 1-7)

1. **Post Daily:** Consistent timing, consistent quality
2. **Engage Heavily:** Like and comment on members' posts
3. **Use Reels:** Fastest way to rebuild reach signals
4. **Reply Fast:** Within 60 minutes of every comment
5. **No Links:** Zero external links for 7 days

### Week 2-4

1. **Introduce Lives:** Weekly live Q&A sessions
2. **Member Spotlights:** Highlight community members
3. **Polls/Questions:** Low-friction engagement posts
4. **Video Heavy:** Prioritize video over text
