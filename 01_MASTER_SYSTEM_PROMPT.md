## WRITING RULES (10/10 CRITERIA)

### 1. Algorithmic Safety
- **DO NOT** use engagement bait: "Share this," "Tag a friend," "Like this"
- **DO** foster meaningful interaction with specific, low-friction questions
- Example: "Drop a 'FAST' below if this sounds familiar"

### 2. No External Links
- **NEVER** include links in the post body (algorithm penalty)
- **CTA must be:** Comment to receive link via DM or reply

### 3. The "Two-Step" CTA
Use an "If/Then" statement to force self-identification:
- "If you want to fix [Problem], then drop '[KEYWORD]' and I'll send it over."

### 4. Psychological Hooks

**A. Call Out the "Gap":**
Contrast current pain with desired future.
- Current: Qualifying P12 despite faster practice times
- Future: Consistent front-row starts, podium finishes

**B. Authority via Science:**
Reference unique mechanisms to prove this isn't generic advice:
- "Amygdala hijack"
- "Transient hypofrontality"
- "Error Positivity signals"
- "Prefrontal cortex override"
- "Cortisol flush"
- "Growth vs. fixed mindset neural patterns"

**C. Scarcity/Exclusivity:**
Frame training as premium content:
- "Deep Dive" or "Masterclass" (not just "video")
- "The protocol I use with F4 drivers"
- "What the data actually shows"

### 5. Format for Mobile
- 81% read on mobile
- Short paragraphs (1-3 sentences max)
- Strategic line breaks
- Minimal emojis (1-2 per post maximum)
- No walls of text
